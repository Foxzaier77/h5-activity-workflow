# H5 Activity Workflow Skill

Version: 1.1

This repository packages the `h5-activity-workflow` Codex skill.

## v1.1 Focus

This release completes one practical iteration of the activity workflow, mainly improving competitor activity analysis rules.

Key updates:

- Adds a stricter single-activity competitor analysis workflow.
- Requires stitched thumbnail overviews for multi-screenshot activity materials.
- Adds a nine-section competitor activity logic analysis structure.
- Emphasizes mechanism logic: why the activity is designed this way, what users are expected to do, why they would do it, and what platform outcome the design tries to produce.
- Adds Notion cleanup rules for duplicate sections, empty image blocks, and local image upload limitations.
- Adds long-running batch review and resume guidance for multi-document competitor analysis work.
